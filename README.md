# JARVIS AI Assistant 🤖

## Overview
JARVIS is a voice-controlled AI assistant built using Python that automates system and browser tasks.

## Features
- Voice command execution  
- Open websites automatically  
- Control media playback  
- Perform system operations  

## Tech Stack
- Python  
- Selenium  
- SpeechRecognition  
- pyttsx3
