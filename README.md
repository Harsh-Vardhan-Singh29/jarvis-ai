# JARVIS AI Assistant 🤖

## Overview
JARVIS is a voice-controlled AI assistant built using Python that automates system and browser tasks. It allows users to interact with their system using voice commands and perform tasks efficiently.

## Features
- Execute commands using voice input  
- Open websites automatically  
- Automate browser actions using Selenium  
- Control media playback  
- Perform basic system operations  

## Tech Stack
- Python  
- Selenium  
- SpeechRecognition  
- pyttsx3  

## Project Structure
jarvis-ai/

## How to Run

1. Clone the repository:
   git clone: https://github.com/Harsh-Vardhan-Singh29/jarvis-ai

2. Install dependencies:
   pip install -r requirements.txt

3. Run the project:
   python main.py
